F-15E WSO Long Throw Stick

High-fidelity simulation component for F-15E WSO training

WSO Long Throw Stick Features:

  • Grip shape is an accurate replication of flight grip
  • Switches have the same form and function as flight grip
  • Reconfigurable joint at the base of the grip allows for grip to be changed to that of a different aircraft quickly
  • Floor Mounted
  • Grip has electrical interface via 37 Pin Mil-Spec Connector
  • Mechanism spring forces simulate aircraft forces
  • All I/O is addressable via USB or Ethernet using Bugeye's BTIO50 card
  • Unit receives power over USB
  • Fidelity: FC2000

Part Number: 20B120693-1001
