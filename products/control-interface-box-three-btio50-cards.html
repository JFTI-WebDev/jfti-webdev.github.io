Control Interface Box (Three BTIO50 Cards)

Professional interface solution for flight simulation equipment

Control Interface Box Features:

  • J.F. Taylor remote / standalone interface card enclosure.
  • Die cast aluminum construction for durability.
  • Includes three BTIO50 USB/Ethernet controller cards.
  • Prewired to three 55-pin female bulkhead connectors.
  • Designed to mate with J.F. Taylor grips or controls.
  • Receives power via USB port on each I/O card.
  • Fidelity: FC2000
