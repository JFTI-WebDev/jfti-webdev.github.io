EO-IR Hand Control Unit

Hand Control Unit Features:

  • Grip shape is an accurate replication of flight grip
  • Switches have the same form and function as flight grip
  • Mounting is via 4 threaded holes in the bottom of the grip
  • Electrical interface via 27 Pin Connector and remote I/O box containing the J.F. Taylor BTIO50 interface card
  • Fidelity: FC2000

Part number: 20B120464-1001
